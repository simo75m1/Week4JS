Reflections

The concept async wait is that it's a more mordern way to work with asynchronous code in js. It is built on promises, just like chaining .then() calls, but makes it more readable.
It also makes error handling easier, as it allows you to use try catch inside the method and send any error messages from the backend, to the console or html page.

I have used it in my findCar methods in findCar.js and editCar.js. If the server returns a false on response.ok when calling the fetch, the error is instead sent out.
