For this exercise i used a simple fetch to get all the beers on load, and save the beers in a local variable for js to use. Then i add an eventlistener to the button, that uses a "filterBeers" method.
The filterBeers method simply grabs the `abv` from the input, uses .filter method on that local array of beers, formats it to a table, and places it in the table innerHTML.
