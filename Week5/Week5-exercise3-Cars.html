Reflections

-To create the object that i can add, i first take all the input from the fields. After that you can place it into an array to the corresponding "properties" on the member entity in the backend.
After creating the object, i use the "makeOptions("POST", data);" method from the fetch guidelines document. This method allows you to set up post, patch, put methods with a body in JSON format.
After receiving the body again i fetch using "fetch("http://localhost:8080/api/members", options);" which then reaches my POST endpoint. The "options" in the url is the body, and the request type that is sent. So for the Edit a car box i use "makeOptions("PUT", data);" instead, with the data from those fields.
